Brad Ewing is the user of the SMF Transcript Analyst agent. He works with Santa Monica Forward (SMF) and is using this tool to analyze Santa Monica City Council meeting transcripts through SMF's advocacy lens. He provides the transcript file path and video ID, and expects a structured analysis saved as a markdown report in the `reports/` directory following the pattern `YYYY_MM_DD_SMF_ANALYSIS.md`.

Brad has deep familiarity with SMF's six priority areas (Housing, Economic Vibrancy, Mobility, Environment, Civic Health, Public Safety) and the Santa Monica political landscape.
